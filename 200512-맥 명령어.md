## Mac os 명령어 정리

#### Web Storm

- 코드 복제 : command + G
- 코드 위, 아래로 이동 : option + shift + ↓↑
- 같은 단어 선택 : control + G
- 같은 단어 모두 선택 : control + command + G
- 단어 단위로 선택 늘이고 줄이기 : option + shift + ←→
- 커밋 창 열기 : command + K



#### Mac

- 특수문자 : control + command + space
- 화면 캡처 : command + shift + 4
- 뭔가 검색창 : command + space
